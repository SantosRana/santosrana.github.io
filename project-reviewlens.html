    How It Works

    1. User enters a restaurant name or review text
    2. The system fetches reviews using the Naver Blog Search API
    3. KcELECTRA predicts aspect mentions and sentiment polarity
    4. Results are aggregated into FOOD, PRICE, SERVICE, and AMBIENCE
    5. Qwen 2.5 generates explanations, summaries, and restaurant comparisons
    6. Streamlit displays sentiment cards, and detailed keyword breakdowns

ReviewLens

AI-Powered Restaurant Review Analysis System

Project Overview

ReviewLens is an intelligent restaurant review analysis platform that leverages Aspect-Based Sentiment Analysis (ABSA) and local Large Language Models to extract meaningful insights from Naver Blog restaurant reviews. The system automatically identifies key aspects (Food, Price, Service, Ambience) and determines sentiment polarity, providing restaurant owners and customers with actionable intelligence.

Key Features

  • Aspect-Based Sentiment Analysis (ABSA) for restaurant reviews
  • Real-time Naver Blog review fetching and processing
  • Interactive Streamlit dashboard with visual analytics
  • Chat-style interface with structured AI-generated summaries
  • Detailed breakdown of positive and negative keywords for each aspect

Tech Stack

Python Streamlit KcELECTRA Qwen 2.5 7B Ollama PyTorch Transformers Naver API Pandas ABSA

How It Works

  1. User enters a restaurant name or review text
  2. The system fetches reviews using the Naver Blog Search API
  3. KcELECTRA predicts aspect mentions and sentiment polarity
  4. Results are aggregated into FOOD, PRICE, SERVICE, and AMBIENCE
  5. Qwen 2.5 generates explanations, summaries, and restaurant comparisons
  6. Streamlit displays charts, sentiment cards, and detailed keyword breakdowns
